üìä Data Synthesization

  • Access synthetic patient data: Click on Synthetic Data in the navigation bar to see the list of synthetic patient files.
  • Compare synthetic and real data: Select a synthetic patient file, and compare synthetic images (top carousel) with real patient images (bottom carousel).
  • Show/Hide Real Images: Use the Show Second Carousel checkbox to toggle the visibility of real images for comparison.
  • Sync Carousels: Enable the Sync Carousels option to synchronize both carousels, ensuring that the real images update while navigating through synthetic images.
  • For clinicians and researchers: This feature is particularly useful for validating synthetic data in clinical settings or training machine learning models.

üìâManual annotation

  • Use the draw function to highlight specific areas and the print function to save the marked sections as a PNG or CSV file on your computer.
  • Use the erase function to erase marked lines.
  • Use the ruler to measure the distance between two points.
  • Use the pen tool to highlight the desired areas, then click the save icon to store the data along with your selected file on your computer.
  • Use interactive features like zoom, pan, and window/level adjustment for better image viewing.

üß†Regional Analysis

  • Access the Regional Analysis page: Click on Regional Analysis in the navigation bar to perform regional analyses of perfusion data based on selected parameters.
  • Patient Category: Choose from LVO (Large Vessel Occlusion), NON_LVO (Non-Large Vessel Occlusion), or All (patients from both groups).
  • Value 1: Choose one of the perfusion metrics: MTT, COV, CTH, TTP, CBF, or Tmax.
  • Value 2: Select a second perfusion metric for comparison, which includes the same options as Value 1.
  • Dimensionality: Choose the dimensionality of the images‚Äî2D, 3D, or Both.
  • Type: Specify whether the figures represent Re-canalized, Non-Re-canalized cases, or neither.
  • Visualize: After selecting all the parameters, click the Visualize button to generate figures based on your chosen criteria. These figures will appear in the preview fields for easy comparison.

Frequently Asked Questions

  • Who can access patient data? Only authorized medical professionals with an account.
